Questions

  1. The server responds to the client's requests, retrieves, and transmits files to the client.
  2. The client locates and requests web pages through a browser from a server. Then it renders the web page in the browser.
  3. An IP address is an identification every device connected to the Internet has, which lets it communicate with other devices.
  4. DNS stands for Domain Name Server which is a computer that maps websites symbolic names (URLs) to their IP addresses.
  5. The Internet is the worlds largest network of connected devices, while the WWW is an application used on the Internet and a system of interconnected hypertext documents.
  6. HTML is a markup language for documents on web pages, and is used to structurize information and to describe the web page for a browser to read.

Questions 2

  1. Inline CSS styling overrides style elements and external stylesheets because it is more local and therefore more specific.
  2. You can use #id to mark a specific paragraph you want to have a certain styling. Classes can be used when you want e.g. all textboxes on your page to have the same styling.
  3. RGBA stands for "Red, Green, Blue, Alpha", where alpha indicates opacity. RGBA can express essentially all colors, including their different opacities.
  4. We put CSS files in the head because then the styling loads first, making things appear on the page so that it's not blank until everything is loaded. It also prevents re-rendering of the page when the CSS has finally loaded.
  5. To match all the p elements in the HTML, you only need the name "p" for all the paragraphs to be selected.
